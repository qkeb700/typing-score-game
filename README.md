## 개발환경
- Visual Studio Code

## 프로젝트 개요
- CSS를 사용하여 기본 화면 디자인
- Javascript를 사용하여 화면에 색이 랜덤으로 변하는 알파벳이 나타나게 구현
- Javascript로 이 나타난 알파벳과 같은 알파벳을 키보드로 타이핑하면 점수가 올라가게 구현
